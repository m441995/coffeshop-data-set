# Monthly Financial Dashboard

📊 **Excel Dashboard (Power Query + PivotTables + Power Pivot)**  
Analyzing monthly revenue, payments, and delivery performance.

## 📈 Key Insights
- Total Revenue (Delivered Orders)
- Expected Revenue (Approved Orders)
- Payments Received & Reconciliation %
- Late Deliveries Count & Average Delay
- Average Order Value

---

## 🧭 How to Use
1. Download the Excel file from the link above.
2. Open `Excel_Dashboard.xlsx`.
3. Go to **Data → Refresh All** to update Power Query connections.
4. Use slicers for filtering by month, order status, and payment type.
